# Cherry Picked

Introduction:

Cherry Picked is full stack application that serves as a beauty/cosmetics review site, similar to Yelp, while providing cosmetics inspiration for customers to, simliar to Pintrest. Brand users can get, post, edit and delete products while customer users get, post, edit, and delete reviews for those products. 

Technologies Used:
Backend: Flask, Python, SQLAlchemy
Frontend: ReactJS, Redux, NodeJS
